## ✨ About the Project

**Glow Theory** is a skincare-themed website focused on self-care, natural beauty, and sustainability.
It provides users with skincare insights, tips, and interactive features to enhance their routine.

---

## 🚀 Features

* 🏠 **Home Page** with image slider and featured sections
* 📖 **About Us** with brand mission and vision
* 📰 **News & Events** for trends, updates, and offers
* ✍️ **Blogs** with skincare tips and guides
* 🧠 **Skin Quiz** for personalized recommendations
* 🖼️ **Gallery** showcasing products visually
* 📍 **Contact Page** with form and Google Maps integration
